Email Validator Project

Django + Celery + RabbitMQ + Redis + Postgres, running on Docker

About

This project is a backend system built with Django and Django REST Framework. It validates emails asynchronously using Celery workers, RabbitMQ for messaging, and Redis as a broker and cache. The database is managed with PostgreSQL.

Running Locally

  1. Clone the repository
  2. Install Docker & Docker Compose
  3. Run docker compose up --build
  4. Open http://localhost:8000

API Usage

Example request with Postman: 

POST http://localhost:8000/api/validate-emails/
{
  "emails": ["demo1@example.com", "demo2@gmail.com"]
}
